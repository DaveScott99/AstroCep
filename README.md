# AstroCep

## Sobre o projeto

Web App que faz cadastramentos de endereços válidos consumindo a API pública ViaCep.

- Link do projeto: https://astro-cep-javascript.vercel.app/

## Tecnologias utilizadas

### Back end

- `Javascript`
- `HTML5`
- `CSS3`
